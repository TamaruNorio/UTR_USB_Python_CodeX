## ログ記録時の注意

- 16進数表記を統一すること。
- スペース有無を揃えること。
- STX/ETX/SUM/CRを確認すること。
- ACK/NACKのどちらかを明記すること。
- 実機固有情報、顧客名、シリアル番号などはマスクすること。
- Python側ログは、Python実機通信の手順と接続条件を確認した後に記入すること。
