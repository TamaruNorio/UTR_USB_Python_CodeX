# USB実機通信前チェックリスト

USB実機通信を行う前に、以下を確認します。実機通信は手順と接続条件を確認したうえで行います。

## PC環境

- [ ] Pythonバージョンを確認した
- [ ] pyserial が導入されていることを確認した
- [ ] COMポートが Windows に認識されていることを確認した
- [ ] 使用する COMポート番号を控えた

## UTR機器と周辺準備

- [ ] UTR機器の電源が入っていることを確認した
- [ ] アンテナが接続されていることを確認した
- [ ] RFタグを準備した
- [ ] UTRRWManagerで事前に通信できることを確認した

## 通信前の確認項目

- [ ] ROMバージョンを確認した
- [ ] 送信出力設定を確認した
- [ ] 誤って送信出力を書き換えないことを確認した
- [ ] Python側ログと UTRRWManager側ログを比較できる準備をした

## 作業ルール

- [ ] `src/utr_usb_sample.py` を変更しない
- [ ] `src/utr_usb_sample.py` を実行する前に手順と接続条件を確認する
- [ ] USB実機通信は確認済みの手順に沿って行う
- [ ] 公開前に差分、テスト結果、機密情報の有無を確認する
