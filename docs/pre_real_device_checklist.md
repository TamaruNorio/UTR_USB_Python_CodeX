# USB実機通信前チェックリスト

USB実機通信を行う前に、以下を確認します。実機通信は明示許可後のみ行います。

## PC環境

- [ ] Pythonバージョンを確認した
- [ ] pyserial が導入されていることを確認した
- [ ] COMポートが Windows に認識されていることを確認した
- [ ] 使用する COMポート番号を控えた

## UTR機器と周辺準備

- [ ] UTR機器の電源が入っていることを確認した
- [ ] アンテナが接続されていることを確認した
- [ ] RFタグを準備した
- [ ] UTRRWManagerで事前に通信できることを確認した

## 通信前の確認項目

- [ ] ROMバージョンを確認した
- [ ] 送信出力設定を確認した
- [ ] 誤って送信出力を書き換えないことを確認した
- [ ] Python側ログと UTRRWManager側ログを比較できる準備をした

## 作業ルール

- [ ] `src/utr_usb_sample.py` を変更しない
- [ ] `src/utr_usb_sample.py` を実行する前に明示許可を得る
- [ ] USB実機通信は明示許可後のみ行う
- [ ] `git commit`、`git push`、PR作成、merge は明示許可後のみ行う
