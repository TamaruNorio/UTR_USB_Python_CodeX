# UTRRWManager 操作手順

この手順書は、USB実機確認を行う前に UTRRWManager 側で確認する内容を整理するためのものです。今回は手順書作成のみで、UTRRWManagerの実操作やPython側からの実機通信は行いません。

## 事前準備

| 準備物 | 確認内容 | 記録欄 |
|---|---|---|
| UTR機器 | 対象機器が手元にあること | 実機確認時に記入 |
| USBケーブル | PCとUTR機器を接続できること | 実機確認時に記入 |
| アンテナ | 接続状態を確認できること | 実機確認時に記入 |
| RFタグ | Inventory確認用のタグを準備すること | 実機確認時に記入 |
| Windows PC | UTRRWManagerを実行できること | 実機確認時に記入 |
| UTRRWManager | 起動できること | 実機確認時に記入 |
| COMポート | Windows上で認識されていること | 実機確認時に記入 |

## UTRRWManagerで確認する順番

| No. | 手順 | 記録すべき項目 | Python側比較用ログ |
|---:|---|---|---|
| 1 | COMポート選択 | COMポート番号、表示名 | Pythonで使用予定のCOMポート |
| 2 | 接続確認 | 接続成功/失敗、エラー表示 | 接続条件、通信速度 |
| 3 | ROMバージョン確認 | ROMバージョン、ACK/NACK | 送受信ログ、応答内容 |
| 4 | コマンドモード確認 | モード移行結果、ACK/NACK | 送受信ログ、応答内容 |
| 5 | 送信出力読み取り | 読み取った送信出力値 | 送受信ログ、出力値 |
| 6 | 周波数チャンネル読み取り | チャンネル番号、周波数 | 送受信ログ、チャンネル値 |
| 7 | Inventoryパラメータ確認 | 取得したパラメータ | 送受信ログ、パラメータ |
| 8 | Inventory実行 | 実行結果、タグ検出有無 | 送受信ログ、Inventory応答 |
| 9 | 読み取り枚数確認 | 読み取り枚数 | Python側の読み取り枚数 |
| 10 | PC/UII確認 | PC/UIIデータ | Python側のPC/UIIデータ |
| 11 | RSSI確認 | RSSI値 | Python側のRSSI値 |
| 12 | NACK/エラー確認 | NACKコード、エラー表示 | Python側のNACK解析結果 |

## 保存すべきログ

- UTRRWManagerの送信コマンドログ
- UTRRWManagerの受信レスポンスログ
- ROMバージョン確認結果
- 送信出力読み取り結果
- 周波数チャンネル読み取り結果
- Inventory実行結果
- NACKやエラーが出た場合の表示内容

## 注意点

- まだPython側から実機通信しないこと。
- 送信出力を書き換えないこと。読み取りと書き込みを混同しないこと。
- アンテナ未接続状態での確認は、明示的にテストするときだけ行うこと。
- 実IP、顧客名、シリアル番号などを記録する場合はマスクすること。
- 実機ログや実測値は、この手順書には作らず、実機確認時に記入すること。
